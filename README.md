# MedLink — Clinic Appointment Management System

MedLink is a full-stack web application that lets patients and doctors manage clinic appointments in real time. Patients browse availability, book visits, reschedule, or cancel, while doctors configure working hours and review upcoming/past visits. Availability checks prevent double booking and enforce lead-time rules so schedules stay conflict free.
